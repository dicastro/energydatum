¿Quieres un sitio como este?

Es muy fácil, únicamente tienes que seguir estos sencillos pasos:

  2. Clona tu repo
  3. Añade las lecturas de tu contador en formato csv
  4. Ejecuta el script run.cmd
  5. Haz push de los cambios
